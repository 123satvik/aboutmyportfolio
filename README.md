# Satvik Sharma — Data Analyst Portfolio

A clean, dark-themed personal portfolio built with pure HTML, CSS, and JavaScript.

---

## 📁 File Structure

```
satvik-portfolio/
│
├── index.html          ← Main HTML page (structure & content)
│
├── css/
│   └── style.css       ← All styles (variables, layout, animations, responsive)
│
├── js/
│   └── main.js         ← All interactivity (cursor, scroll reveal, nav highlight)
│
└── README.md           ← This file
```

### External resources loaded via CDN (no download needed):
| Resource       | Purpose                        |
|----------------|--------------------------------|
| Google Fonts   | Syne + Space Mono typefaces    |
| Font Awesome   | Skill & contact icons          |

---

## 🎨 Customization

All colors are defined as CSS variables at the top of `css/style.css`:

```css
:root {
  --accent:  #c084fc;   /* light purple — primary accent */
  --pink:    #ffb3c6;   /* light pink   — project titles */
  --accent2: #ff6b35;   /* orange       — subtitles      */
  --accent3: #7c3aed;   /* deep purple  — ring           */
}
```
